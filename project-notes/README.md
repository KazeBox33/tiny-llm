# Project Notes

This folder contains notes written during the Qwen3 inference learning and optimization work.

## Structure

- `resume/`
  - Resume-ready project notes, implementation summaries, and measurable results.
- `benchmarks/`
  - Human-written benchmark reports and performance interpretation.
- `optimization/`
  - Future optimization backlog, bottlenecks, and planned engineering work.

## Raw Benchmark Outputs

Raw benchmark outputs are kept in the repository-level `benchmarks/` folder.

That split is intentional:

- `project-notes/benchmarks/` contains written reports.
- `benchmarks/` contains generated JSON / text benchmark artifacts.
